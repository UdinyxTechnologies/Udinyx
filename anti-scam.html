Anti-Scam Policy

Our Shield Against Exploitation

Scam companies have slick websites, legitimate-looking registrations, and official-looking PDFs. They can copy designs, but they can never replicate real integrity. We maintain a strict zero-tolerance policy against student exploitation through three uncopyable weapons:

  • Zero Fees, Permanently: We will never charge you for registration, documentation, evaluation, or certificate generation. If an organization asks for a single rupee to hand you an "offer letter" or an "LOR," it is a scam. We are 100% free.
  • No Ghost Cohorts: Scam operations leave you stranded with a zip file of generic tasks. Our policy mandates a strict 10:1 student-to-mentor ratio with weekly live touchpoints to ensure you actually ship functional code.
  • Verification Through Code, Not Paper: A PDF certificate has zero industry value if there is no proof of work behind it. Every single certificate we issue is bound to a public GitHub project repository showing your exact code contributions and commits.
